# MS Word Template 초안 준비 및 저장

## MS word Template 만들고 저장
1. 템플릿으로 사용할 초안을 만듭니다.
2. 일반문서인 docx로 저장합니다.
3. 입력해야할 부분은 나중에 SharePoint에서 추가합니다(무슨말인지 잘 몰라도 일단 나중에 알게 되십니다).
